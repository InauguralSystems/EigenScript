# Symbols the freestanding profile is ALLOWED to import — the ledger's
# (docs/FREESTANDING.md) HAL roots + the write-once mini-libc/libm surface.
# tools/freestanding_check.sh compiles the runtime (minus main.c) with
# -DEIGENSCRIPT_FREESTANDING=1 and fails if the undefined-symbol set
# contains anything not listed here. Adding a symbol to this list is a
# statement that EigenOS's HAL or the portable mini-libc must provide it.
